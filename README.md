# Blog

Welcome to my personal tech blog! Here, I share insights, tutorials, and experiences from my journey in software programming, cloud computing, Kubernetes, Docker, and DevOps practices.

This project is built with [Hugo](https://gohugo.io/), styled using the [`hugo-blog-awesome`](https://github.com/hugo-sid/hugo-blog-awesome) theme, and containerized with Docker for easy local development and deployment.

### Prerequisites

- [Docker](https://www.docker.com/)
- [Git](https://git-scm.com/)

### Clone the Repository

```bash
git clone https://github.com/abellucci93/blog.git
cd blog
```

### Run with Docker

You can run the blog locally using Docker:

```bash
docker compose up -d
```

Now open your browser and visit: [http://localhost:1313](http://localhost:1313)
